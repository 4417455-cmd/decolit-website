# Рефлексия: дизайн v2 сайта — кириллические шрифты и генеративные фактуры

**Дата:** 2026-06-12
**Время:** 20:30

---

## Какая задача была поставлена

Пользователь: сайт запущен, но «пустой, без картинок, с плохими шрифтами и странный» — сделать дизайн уровня «вау».

## Как я её решал

1. Диагностика: главные причины — Fraunces и Hanken Grotesk **не содержат кириллицы** (весь русский текст падал в системную Georgia/Arial), а «фактуры» были плоскими CSS-градиентами.
2. Самохостинг шрифтов с кириллицей: Prata (дисплейная антиква) + Golos Text (вариативный), woff2 ~123 КБ суммарно, без внешних CDN.
3. Генеративные SVG-фактуры 7 шт (`assets/img/tex/`): `feTurbulence` + `feDiffuseLighting` + тонирование `feComponentTransfer` — реалистичный рельеф штукатурки при ~2 КБ на файл. Итерации через скриншоты headless Chrome.
4. Редизайн style.css: тёмный hero с Ken Burns, тикер материалов, тёмная секция-манифест, карточки-«экспонаты», футер с крупным вордмарком.
5. JS: рабочий фильтр галереи (data-cat/data-filter), count-up счётчики.
6. Скриншот-проверка десктоп/мобайл/страница услуги, исправления (перенос ₽, слишком тёмные секции, мобильная типографика).

## Решил ли

- [x] Да, полностью (в рамках шаблона — контент остаётся заглушками)

## Насколько эффективно, что можно было лучше

- Потерял время на борьбу с headless Chrome: фрагментные якоря + reveal-анимации дают пустые скриншоты. Рабочая схема: отладочная копия страницы с `.reveal{opacity:1!important}` + ограничение высоты hero + один высокий кадр, нарезаемый sips.
- Первые версии «прожилок» через ridge-таблицу на `feTurbulence` дают «леопардовые» кольца; лечится сильно анизотропной частотой (0.001 против 0.01) + поворотом слоя.

## Как было и как стало

| До | После |
|---|---|
| Заголовки в системной Georgia (нет кириллицы у Fraunces) | Prata + Golos Text, самохостинг, надёжно из РФ |
| Плоские градиенты вместо фактур | 7 генеративных SVG-фактур с рельефом и светом |
| Светлый невыразительный hero | Тёмная бронзовая стена, крупная антиква, Ken Burns |
| Фильтры галереи — декорация | Рабочий фильтр + анимированные счётчики |
| Шрифты с Google Fonts CDN | Ноль внешних запросов |

## Что пробовал и не сработало

- [x] Попытка: скриншоты секций через якоря `#works` в headless — Почему не сработало: фрагментная навигация в headless не скроллит/не дорисовывает; ушёл на полностраничный кадр с дебаг-CSS.
- [x] Попытка: `--force-prefers-reduced-motion` чтобы раскрыть `.reveal` — Почему не сработало: флаг не подействовал на media query в этой сборке Chrome 149.
- [x] Попытка: прожилки мрамора ridge-таблицей на изотропном `feTurbulence` — Почему не сработало: получаются замкнутые «кольца» (камуфляж), а не прожилки; нужна анизотропия + поворот.

## Что записать в ai-clone/feedback/

- Правило: для русскоязычных сайтов проверяй кириллицу в выбранных шрифтах ДО подключения (Fraunces, Hanken Grotesk, Spectral — без кириллицы). Why: весь текст молча падает в системный фолбэк — «плохие шрифты». How to apply: сверять поддержку subsets на Google Fonts / в спецификации шрифта; для РФ — самохостинг woff2.
