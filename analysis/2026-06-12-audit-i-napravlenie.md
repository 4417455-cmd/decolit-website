## ГЛАВНЫЙ ВЫВОД (сходятся все 4 агента)

Текущий сайт — **пассивный каталог из конструктора**, а не продающий инструмент. Парадокс: сайт про **тактильные поверхности почти не показывает поверхности**. Премиум заявлен словами, но не подтверждён ни визуально, ни социально.

На рынке Москвы **никто из конкурентов не сделал галерейный, видео-первый, тактильный сайт с дизайнерской B2B-дверью**. Это незанятая ниша — место для отрыва.

### 4 ключевые дыры

1. **Нет визуала.** На страницах услуг — ноль фото работ. Галерея — безымянный набор фоток внизу.
2. **Нет доверия.** Отзывов нет нигде. Кейсов нет. Сертификаты только в FAQ. Гарантия 3 года зарыта в список. Нет «лиц» компании.
3. **Слабый оффер.** H1 = название категории, а не оффер. УТП — шум («индивидуальный подход / надёжность»). Сильные активы (20 лет, своя бригада, образец на следующий день, фикс-смета) спрятаны.
4. **Нет захвата лида и сегментации.** Один тип формы, нет калькулятора (хотя цены за м² открыты!), квиза, записи на замер. B2C и B2B не разделены — дизайнеру (самая маржинальная аудитория) посвящена одна строчка.

### 3 вектора отрыва (нет ни у кого в РФ-нише)

1. **Видео-первый сайт** — встроенный timelapse нанесения каждой фактуры. Дёшево, сильный «вау», показывает ремесло.
2. **Дизайнерская B2B-дверь** (образец San Marco) — выкрасы под проект, условия партнёрства/вознаграждения, прайс.
3. **AR/визуализатор фактуры на стене** — флагманский «вау», вторым этапом (дороже).

# Отчёт 1 — UX / Конверсия (CRO)

Изучено: главная, `/mikrocement/`, `/venezianka`, `/gallery`, `/article1`.

**Вердикт:** сайт работает как пассивный справочник цен, а не машина продаж. Слой доверия и слой захвата лида почти отсутствуют.

## Структура страницы
Текущий порядок: шапка → баннер «Заказать звонок» → H1 → «О нас» (3) → услуги (10 плиток) → «Почему доверять» (6) → форма → процесс (4 этапа) → FAQ (4) → портфолио → статьи (3) → форма → подвал.

Проблемы: первый экран не продаёт (H1 = категория товара, нет ключевого визуала); дублирование блоков доверия («О нас» и «Почему доверять» — про одно); портфолио почти внизу (в визуальной нише должно быть в верхней трети); отзывов нет вообще.

**Рекомендуемый порядок:** Hero (оффер + крупное фото + 2 CTA) → галерея-витрина с фильтром → виды покрытий с ценой «от» → калькулятор/квиз → блок доверия (цифры, сертификаты, гарантия 3 года крупно, логотипы брендов) → отзывы + видео + кейсы → процесс (4 этапа — оставить, он хороший) → блок для дизайнеров/прорабов → FAQ → финальная форма + контакты + карта.

## Конверсионные элементы
CTA слабые и однообразные («Заказать звонок», «Оставить заявку» — нагрузочные действия). Сильная — «Рассчитать стоимость материала и работ», её делать главной. На страницах услуг три кнопки подряд = паралич выбора.

Чего критически не хватает:
- **Калькулятор стоимости** (площадь × покрытие = вилка) — все цены за м² открыты, собрать легко. Самый недорогой и конверсионный апгрейд.
- **Квиз** («Подберём покрытие + рассчитаем за 1 мин») — тёплые лиды.
- **Запись на бесплатный замер** с календарём.
- **Лид-магнит**: бесплатный образец почтой, PDF «Каталог фактур 2026», подборка интерьеров.
- **Плавающий виджет мессенджеров** + липкая кнопка звонка на мобайле (сейчас Telegram спрятан в подвале).

## Доверие — самый слабый блок
Отзывов нет нигде. Кейсов нет (галерея — безымянные фото). На страницах услуг нет ни одного фото работы. Нет «до/после». Гарантия 3 года зарыта. Сертификаты только упомянуты. Нет «лиц» компании.
Усилить: видеоотзывы клиентов и дизайнеров; 6–10 развёрнутых кейсов (объект, м², покрытие, дизайнер); счётчики (лет / объектов / м²); логотипы брендов материалов; сертификаты галереей; раздел «Команда».

## CJM — нет сегментации
- **B2C (частник):** боится «дорого/сложно/как будет выглядеть/испортят стену». Нужны: фото интерьеров, калькулятор/квиз, образец, отзывы таких же, понятный процесс. Сейчас упирается в цены без визуала.
- **B2B (дизайнер/прораб):** нужны прайс/каталог для скачивания, условия партнёрства и вознаграждение, скорость, выкрасы под проект, образцы оптом. Сейчас — одна строчка, скидок и условий нет. Упущена самая маржинальная аудитория.

## Мобайл (строить mobile-first)
Липкая кнопка звонка внизу; одна основная CTA вместо трёх; таблицы цен → карточки (избежать горизонтального скролла); свайпабельная галерея с быстрыми превью.

## Скорость
Конструктор + тяжёлые фото без оптимизации → риск высокого LCP. Заложить lazy-load, WebP/AVIF, CDN, резервировать размеры (CLS). Таргет: Performance mobile 80+, LCP < 2.5с, CLS < 0.1.

7. Квиз — высокий / средний

# Отчёт 2 — Дизайн / Арт-дирекшн

**Концептуальная проблема:** сайт про тактильные поверхности почти не показывает поверхности. Текста и «преимуществ» больше, чем материала.

## Слабые места текущего визуала
- Типографика без характера (один sans, заголовки = просто жирнее текста).
- Фото вторичны, нет ни одной крупной макросъёмки текстуры во весь экран.
- Сетка карточная, «магазинная» (плитки иконка+заголовок+текст).
- Иконки-клипарт удешевляют.
- Тесные отступы, мало воздуха.
- Цвет нейтральный, но безликий — нет фирменного материального акцента.

РФ-конкуренты (microdec, decorazza, velesart) страдают тем же — «информационные, не люксовые». **Это рыночная щель.** Эталон брать у итальянцев (Oikos, San Marco) и архитектурных студий с Awwwards.

## Эстетика ниши
Продукт — это свет на поверхности. Принципы: фото = 80% дизайна (full-bleed, чередование макро текстуры ↔ интерьер в контексте); скользящий боковой свет (техтребование к съёмке); палитра «земли и камня»; минимум интерфейса, максимум материала; медленный спокойный ритм.

## Дизайн-направление

### Палитра (тёплый нейтрал)
| Роль | Hex |
|---|---|
| Фон основной | `#F4F1EC` / `#EDE8E0` (известковый офф-вайт) |
| Текст основной | `#1C1A17` (почти-чёрный, тёплый подтон) |
| Графит | `#3A3733` |
| Тауп/глина | `#A89B8C` |
| Песок | `#C9B79C` |
| Акцент (терракот/глина) | `#9C6F4E` или холоднее `#6E6A60` |
| Карточный фон | `#FFFFFF` / `#FBF9F5` |

Граница: один акцентный цвет. Глубина — от фактуры фото, не от палитры.

### Типографика (пара serif + sans)
- **Заголовки:** `PP Editorial New` (первый выбор); альт. `GT Sectra`, `GT Alpina`, `Canela`; бюджетно/Google: `Fraunces` (variable, ось opsz) или `Cormorant`.
- **Текст/интерфейс:** `Söhne` / `Neue Montreal`; бюджетно/Google: `Inter`, `Geist`, `Manrope`. Капс с трекингом `0.08em` для категорий.
- Крупные дисплейные H1 (clamp 48–96px), межстрочный 1.05–1.1 в заголовках, 1.6 в тексте.

### Сетка
12 колонок, используется асимметрично/галерейно. Full-bleed фото вперемешку с текстом у края. Много негативного пространства. Ширина текста ≤720px, фото — на весь вьюпорт.

### Стиль фото
Реальные проекты, боковой свет, тёплый колор-грейдинг (убрать синеву). 2 кадра на покрытие: макро + интерьер. Никаких стоков и 3D-рендеров вперемешку с фото.

### Три концепции
- **А. «Минимализм-галерея»** (рекомендую как основу) — сайт как онлайн-галерея материалов, покрытия = экспонаты с табличками. Лучший баланс «вау»/конверсия, отстраивает от всех РФ.
- **Б. «Тёплый люкс»** — темнее, графит/тауп фон в hero, акцент терракот, ощущение бутика. Дозировать тёмные секции.
- **В. «Архитектурный»** — строгая сетка, технический sans, упор на чертёжность и B2B. Холоднее, для B2C слабее.

**Вывод:** база — А (галерея) с тёплой палитрой; для hero и «философии» вставить приём Б (одна тёмная тактильная секция).

## Эффекты — чёткая граница
Анимировать только `transform` и `opacity`.
- **Усиливают:** fade/slide-up reveal (300–500ms, ease-out, смещение 16–24px); тонкий Ken Burns zoom на hero (8–12с); hover scale 1.03 + проявление подписи на карточках; умеренный параллакс только на десктопе; прелоадер 1–1.5с (маскирует LCP).
- **Вредят:** тяжёлый full-page параллакс на мобайле; скролл-джекинг; множественные одновременные анимации, авто-карусели; анимация теней/`width`/`height`/`top`; эффекты ради эффектов.
- Обязательно: `prefers-reduced-motion`, passive listeners, lazy-load, AVIF/WebP + srcset, зелёный Lighthouse.

## 5 принципов mood
1. Поверхность — герой. 2. Тёплый минерал, не холодный минимал. 3. Тихая роскошь (воздух, медленный ритм, редакционный serif). 4. Галерея, а не магазин. 5. Эффекты служат материалу; скорость и конверсия выше любого трюка.

**Итог:** концепция «галерея фактур» — тёплая известковая палитра, `PP Editorial New`/`Fraunces` + `Söhne`/`Inter`, full-bleed макросъёмка при боковом свете, мягкие reveal, максимум воздуха. _(Итальянские бренды — только дизайн-референс чужих сайтов, не позиционирование Деколитпро.)_

## SEO
- **URL и заголовки** — переименовать под запросы с 301: `/mikrocement`, `/venecianskaya-shtukaturka`, `/shtukaturka-pod-beton`, `/shelkovaya-shtukaturka`, `/fakturnaya-shtukaturka`. Title/H1 по формуле «{тип} {в Москве} {цена/под ключ}».
- Не хватает: Title/Description, микроразметки (LocalBusiness, FAQPage, Review/AggregateRating), карты, страниц «Отзывы» и «Портфолио», хлебных крошек.

# Отчёт 4 — Конкуренты и рынок

## Российские конкуренты (проверены, живые)
1. **ДЕСА / mxfloor.ru** — сильнейший референс. Портфолио 120+ с категориями, прозрачный прайс, 100+ фактур, PDF-каталог, 8 отзывов с датами, шоурум. Нет: калькулятора, визуализатора, фильтров портфолио, видео.
2. **DecorLux / cementmicro.ru** — лучшая работа с доверием. Квиз-калькулятор (3 вопроса + 3 образца за 9000 ₽), 350+ проектов, процесс из 8 этапов, гарантия 10 лет, партнёры-дизайнеры, WhatsApp. Нет: фильтров портфолио, прозрачной финальной цены.
3. **MicroDec / microdec.ru** — единственный с рабочим калькулятором (→PDF→корзина), портфолио с фильтрами по помещениям, сертификаты, MosBuild, YouTube. Нет: визуализатора, отзывов, акцента на дизайнеров; фото низкого разрешения.
4. **Loggia / loggia.ru** — прозрачные стартовые цены (микроцемент от 7000 ₽/м², венецианка от 900 ₽/м²), описание технологии, фильтры по характеристикам. Нет: калькулятора, портфолио на странице, отзывов, гарантии.
5. **Decorazza / decorazza.ru** — производитель: видеоуроки, «Школа мастеров», навигация по эффектам/помещениям. Портфолио/отзывы пустые, прайс по авторизации.
6. **Venicolor / venicolor.ru** — итальянские материалы + услуга, глубокий каталог. Цены не показаны, нет отзывов/калькулятора, SVG-заглушки вместо фото.
7. **Мастера Декора, RospisSten** — нишевые подрядчики, ориентир по ценам (венецианка от 3800, микроцемент от 4800 ₽/м²).

## Дифференциация
**Позиционирование:** не «штукатурка под ключ» (как все), а **«авторская студия декоративных поверхностей для дизайнеров интерьеров»**.
1. Раздел для дизайнеров (B2B-дверь) — незанятая ниша в рунете.
2. Видео-первый сайт (timelapse каждой фактуры).
3. AR/визуализатор — вторым этапом.
4. Mood-навигация + эмоциональный нейминг.
5. Прозрачный квиз-калькулятор (→PDF-смета + платные образцы).
6. Доказательная база (фильтруемое HD-портфолио + отзывы + «до/после» + сертификаты + гарантия) — обязательный минимум «догнать лидеров».

Логика: п.6 — догнать ДЕСА/DecorLux; п.1–2 — быстрый дешёвый отрыв; п.3 — флагман вторым этапом.
