# Контентная база сайта Деколит

Сайт сейчас статический: отзывы, цены, карточки покрытий и часть цифр зашиты прямо в HTML/JS.
Эта папка — первый источник правды, который будем постепенно заполнять реальными данными.

Это не серверная база данных. Для текущего сайта правильнее начать с JSON-файлов:

- их легко открыть и править;
- они хранят историю изменений в git;
- из них потом можно генерировать HTML-страницы;
- их можно перенести в CMS или настоящую БД, если сайт вырастет.

## Файлы

- `company.json` — контакты, адреса, реквизиты, доверительные тезисы, процесс работы, FAQ.
- `coverings.json` — покрытия, цены, тексты, статусы подтверждения.
- `reviews.json` — отзывы клиентов и дизайнеров, согласия на публикацию.
- `projects.json` — реальные объекты/портфолио, фото, покрытия, площади, разрешения.

## Главный источник истины

После проверки от 2026-06-16 главным источником считаем `https://decolitpro.ru/`.
Данные, помеченные `confirmed_from_decolitpro`, можно использовать как подтвержденные:

- цены материала и нанесения;
- список покрытий;
- контакты, адреса, реквизиты;
- тезисы доверия, этапы работы и базовый FAQ.

Отзывы на `decolitpro.ru` при проверке не найдены, поэтому текущие отзывы в `reviews.json`
остаются заглушками до отдельного подтверждения.

## Статусы

- `confirmed` — подтверждено, можно публиковать.
- `needs_confirmation` — выглядит правдоподобно, но нужно подтвердить.
- `placeholder` — демонстрационная заглушка, нельзя считать реальными данными.
- `draft` — черновик, можно редактировать.

## Как заполняем

1. Сначала подтверждаем цены: материал, нанесение, сумма материала и нанесения, сложные зоны.
2. Затем подтверждаем отзывы: имя, текст, объект, можно ли публиковать имя/ЖК/фото.
3. После этого заносим реальные объекты портфолио: фото, покрытие, помещение, площадь, год.
4. Только подтвержденные данные переносим на страницы сайта.

## Минимум данных перед публикацией

- цены по 6 основным покрытиям;
- 6-10 реальных работ с фото;
- 3-5 отзывов с разрешением на публикацию;
- реальные цифры компании: стаж, количество объектов, гарантия, адрес шоурума.
