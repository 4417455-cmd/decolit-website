# Деколит — сайт (шаблон)

Шаблон нового сайта студии декоративных поверхностей. Концепция «галерея фактур»
(см. `../analysis/2026-06-12-audit-i-napravlenie.md`).

> ⚠️ **Это шаблон.** Отзывы, цифры (20 лет, 120 объектов и т.д.), названия ЖК и имена
> клиентов — **демонстрационные заглушки**. Фактуры покрытий — генеративные SVG
> (`assets/img/tex/`), а не реальные фото. Всё это заменяется реальными данными перед запуском.

## Как открыть

Просто открыть `index.html` в браузере (двойной клик).
Или поднять локальный сервер из папки `site/`:

```
python3 -m http.server 8000
# затем открыть http://localhost:8000
```

## Что на странице (блоки)

1. Шапка (фиксированная, сворачивается при скролле) + мобильное меню
2. Hero — тёмная фактурная стена, оффер, 2 CTA, ключевые цифры
3. Тикер материалов (бегущая строка)
4. Галерея работ (рабочие фильтры-чипы + плитки с подписями)
5. Манифест — тёмная секция-цитата
6. Покрытия — карточки-«экспонаты» с ценой «от» и тегами
7. Калькулятор стоимости (рабочий: покрытие × площадь → вилка)
8. Блок доверия — анимированные счётчики
9. Процесс — 4 этапа
10. Отзывы — 3 карточки (заглушки)
11. Дизайнерам (B2B-блок)
12. FAQ — аккордеон
13. Финальная форма заявки
14. Футер с крупным вордмарком

## Технические заметки

- Без сборки и зависимостей — чистые HTML/CSS/JS. Шрифты самохостинговые
  (`assets/fonts/`): Prata — заголовки, Golos Text — текст; обе с кириллицей,
  внешних CDN нет (надёжно из РФ).
- Фактуры материалов — генеративные SVG (шум `feTurbulence` + рельеф
  `feDiffuseLighting`), каждая ~2–3 КБ. Реальные фото заменят их позже
  теми же классами `.tex-*`.
- Анимации только `transform`/`opacity`; учитывается `prefers-reduced-motion`.
- Адаптив: брейкпоинты 1020px и 680px, mobile-first дружелюбно.
- Формы и кнопки — без бэкенда (заглушки). Подключение CRM/телефонии — отдельная задача.
