# Картинки сайта

Сюда кладутся реальные изображения, когда они появятся.

## Сейчас (шаблон)

- `grain.svg` — зерно/шум, накладывается на CSS-фактуры (используется классом `.grain`). Это рабочий ассет, не заглушка.

Все «фотографии» покрытий и работ на странице сейчас **нарисованы стилями** (CSS-текстуры: микроцемент, венецианка, бетон, шёлк, замша, фактурная) — внешних фото-файлов пока нет. Это сделано специально: шаблон всегда отображается одинаково и не зависит от наличия фото.

## Когда будут реальные фото

Структура-предложение:

```
assets/img/
├── grain.svg
├── works/        ← фото объектов для галереи (works.lg, works.md и т.д.)
├── materials/    ← макросъёмка фактур для карточек покрытий
├── hero/         ← крупный кадр для первого экрана
└── reviews/      ← аватары/фото клиентов (по согласию)
```

Требования к фото (из дизайн-аудита `analysis/2026-06-12-audit-i-napravlenie.md`):
- боковой скользящий свет, чтобы читалась фактура;
- 2 типа кадра на покрытие: макро текстуры + интерьер в контексте;
- тёплый колор-грейдинг (без синевы);
- отдавать в WebP/AVIF + `srcset`, lazy-load.

Замена CSS-текстуры на фото: в карточке/плитке вместо `<div class="surface tex tex-...">` ставится `<img>` или `background-image` на нужный файл.
